# FrontendAngularTechnicalTest

Requirements
============
* Node v18.19
* Angular ver 18

Instalation
============

Clone repo using ssh (make sure already have ssh on github account)

    git@github.com:mutiaayuriz/frontend-angular-technical-test.git

Install node module

    npm install

atau 

    npm install --force


Running Project
============

For running project you can use:

    npm start
